Hilfe beim Senden einer Nachricht an Emanuel


Schritt 1

Wir drücken die Schaltfläche „Kontaktieren Sie mich“ entweder im oberen Menü oder im Abschnitt „Startseite“.

Nachdem Sie eine der beiden genannten Schaltflächen gedrückt haben, müssen Sie zum Abschnitt "Kontaktieren Sie mich" gelangen.

Schritt 2

Füllen Sie alle Felder im Bereich „Kontakt“ mit Ihren Daten aus.

Schritt 3

Klicken Sie auf den Button mit dem Text "Nachricht senden!", damit die Nachricht Emanuel erreicht.

Du hast es geschafft!

Nachdem Sie auf die Schaltfläche „Nachricht senden!“ geklickt haben, hat Ihre Nachricht Emanuel erreicht.

Wenn Sie eine Seite erreicht haben, die wie das Bild unten aussieht, bedeutet dies, dass Sie die Nachricht erfolgreich gesendet haben.


Der Rücken

Jetzt können Sie zur Hauptseite zurückkehren, indem Sie auf die Schaltfläche mit dem Text „Zurück zur Hauptseite“ klicken.

Herzliche Glückwünsche !

Das hast du sehr gut gemacht!
